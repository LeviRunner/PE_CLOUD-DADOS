STUDY TRACKER - GUIA DE INÍCIO RÁPIDO
=====================================

Parabéns! Você tem a aplicação web Study Tracker pronta para usar.

INÍCIO RÁPIDO (2 minutos)
=========================

Windows:
--------
1. Abra prompt de comando na pasta study-tracker
2. Digite: python -m http.server 8000
3. Acesse: http://localhost:8000

Mac/Linux:
----------
1. Abra terminal na pasta study-tracker
2. Digite: python3 -m http.server 8000
3. Acesse: http://localhost:8000

Pronto! A aplicação está rodando.

COMO USAR
=========

1. ESCOLHA SEU CAMINHO
   - Clique nas abas: Cloud Computing, Ciência de Dados ou Full-Stack Cloud
   - Escolha qual caminho quer seguir

2. MARQUE SEUS CURSOS
   - Clique na caixa de seleção quando completar um curso
   - Seu progresso é salvo automaticamente

3. ACESSE OS LINKS
   - YouTube: Leva direto aos vídeos do YouTube
   - Site: Abre o site oficial do curso

4. ACOMPANHE SEU PROGRESSO
   - No topo da página, veja sua porcentagem de conclusão
   - Cada fase mostra seu progresso individual

5. EXPORTE SEUS DADOS
   - Clique em "Exportar Progresso" para baixar um arquivo JSON
   - Você pode compartilhar isso ou fazer backup

FUNCIONA OFFLINE
================

A aplicação funciona 100% offline:
- Use normalmente sem internet
- Seus dados são salvos localmente
- Quando conectar novamente, tudo sincroniza automaticamente

INSTALAR COMO APP
==================

Desktop (Chrome/Edge):
1. Abra a aplicação
2. Clique no botão "Instalar App" (canto superior direito)
3. Confirm a instalação
4. Agora funciona como um app nativo!

Mobile (Android/iPhone):
1. Abra no navegador
2. Menu → Instalar App (ou "Adicionar à tela inicial")
3. Funciona como app nativo

ATALHOS DE TECLADO
==================

Ctrl+S (ou Cmd+S no Mac)
- Exporta seu progresso em JSON

Ctrl+Shift+C (ou Cmd+Shift+C no Mac)
- Muda para a próxima aba

RESETAR DADOS
=============

Se quiser limpar todo o progresso:
1. Clique em "Limpar Dados"
2. Confirme a ação
3. Tudo volta ao zero

DEPLOY NA INTERNET
===================

Quer compartilhar com amigos? Opções gratuitas:

GitHub Pages:
1. Crie repo no GitHub
2. Coloque os arquivos
3. Settings → Pages → Enable
4. Acesse seu site em: seu-usuario.github.io/study-tracker

Vercel (recomendado):
1. Vá em vercel.com
2. Conecte seu GitHub
3. Deploy automático em cada mudança
4. Seu site ao vivo em minutos!

Netlify:
1. Vá em netlify.com
2. Drag & drop a pasta
3. Seu site ao vivo instantaneamente

Railway:
1. Vá em railway.app e conecte seu GitHub
2. O Railway lê o nixpacks.toml e roda o start.sh
3. A aplicação escuta automaticamente na porta $PORT
4. Cada push gera um deploy novo

PROBLEMA? SOLUÇÃO RÁPIDA
========================

Dados não salvam:
- Abra Console (F12) e veja mensagens de erro
- Limpe localStorage: localStorage.clear()

App não carrega:
- Limpe cache do navegador (Ctrl+Shift+Del)
- Recarregue a página (Ctrl+F5)

Service Worker não funciona:
- Verifique se está em http://localhost ou https
- Abra DevTools → Application → Service Workers

Progresso desapareceu:
- Verifique se não limpou dados acidentalmente
- Se tiver um backup JSON, você pode restaurar manualmente

NÃO TENHO BANCO DE DADOS
========================

NÃO PRECISA! A aplicação funciona totalmente offline:

Todos os dados são salvos no seu navegador (localStorage)
- Funciona sem servidor
- Funciona sem internet
- Funciona para sempre (até você limpar)

Se perder os dados:
- Limpe cache → perde tudo
- Trocar navegador → perde tudo
- Por isso existe o "Exportar Progresso"!

PERSONALIZAÇÕES
===============

Quer adicionar/remover cursos?
1. Abra index.html em um editor de texto
2. Procure a seção que quer modificar
3. Copie/Cole um course-item
4. Altere o nome e links
5. Salve e recarregue

Quer mudar as cores?
1. Abra style.css
2. Procure por :root {
3. Mude as cores (--primary-color, etc)
4. Salve e recarregue
