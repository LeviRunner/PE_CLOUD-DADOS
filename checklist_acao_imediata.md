## 💡 DICAS ESSENCIAIS

### 1. Nunca apenas "assistir" a aula
```
❌ ERRADO: Assista 4h de vídeo, sinta-se cansado
✅ CERTO:  Assista 30 min, pause, code 30 min
```

### 2. "Code along" com o instrutor
- Não copie o código
- ESCREVA o código você mesmo enquanto assiste
- Pause a aula quando precisar

### 3. Sempre salve no GitHub
```bash
git add .
git commit -m "Topico: breve descricao"
git push
```
Faça commits frequentes (1-2 por dia)

### 4. Faça exercícios REPETIDAMENTE
**Semana 1:** Faça 1 exercício de calculadora
**Semana 2:** Faça novamente de memória
**Semana 3:** Faça com variações (converter para dólares, euros, etc)

### 5. Documente TUDO
- Arquivo `APRENDIZADO.md` no repo
- Escreva o que aprendeu cada dia (3-5 linhas)
- Anote bugs que resolveu
- Isso vira seu portfolio!
